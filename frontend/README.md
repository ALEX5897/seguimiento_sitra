

## Tecnologías utilizadas
- Vue 3 + Vite
- Bootstrap 5 (tablas dinámicas, diseño )
- DataTables (tablas interactivas)
- Axios (consumo de API)

## Instalación

```bash
cd frontend
npm install
npm run dev
```

## Uso

El frontend asume que el backend corre en el mismo host/puerto (rutas relativas `/api/...`). Si el backend corre en otro puerto, configurar un proxy en `vite.config.js` o usar URLs absolutas.
